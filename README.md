**QuickCmds**

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.6--1.21.8-brightgreen)
![Fabric](https://img.shields.io/badge/Fabric-Supported-DBB69C?logo=fabric&logoColor=black)

A lightweight client-side mod that adds a customizable radial command wheel to Minecraft — press a key to bring up a ring of up to 12 slots and trigger commands, chat messages, or hotkeys with a single click.

![radial menu](https://cdn.modrinth.com/data/cached_images/ee454d8349b2a1275936ce414b5e6e8a9e2c4667.jpeg)

**Features**

Radial Quick Panel — hold or press a key to open a circular menu, hover a segment and click to execute

Three slot types
- Command — run one or several /commands (one per line, with configurable delay between them)
- Chat message — send preset text/messages instantly
- Hotkey — trigger a custom key combination

Fully customizable appearance
- Adjustable wheel size, outer/inner radius and dead zone
- Custom segment color, text color and transparency (normal and hover states)
- Icon picker — choose from built-in icons or any Minecraft item texture

Two independent keybinds — one to open the command wheel, one for a separate quick settings panel

In-game settings GUI — no config file editing needed, everything is configurable through tabs (Commands / Settings / Appearance)

Client-side only — works on any server, no server-side installation required

Localized in 13 languages: English, Russian, Ukrainian, German, French, Spanish, Polish, Turkish, Japanese, Korean, Chinese, Portuguese (BR), Kazakh

How it works

1. Press the Quick Panel key to open the radial wheel
2. Hover over a slot to see its tooltip
3. Left-click to run it, right-click to delete/edit
4. Use the Settings key to configure slots, appearance, and keybinds without leaving the game

Perfect for survival multiplayer, creative building, server administration, or just speeding up repetitive command/chat workflows.

Compatibility

Client-side mod — safe to use on vanilla servers, no server installation needed.
